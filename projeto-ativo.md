# Projeto Ativo

## Roteiro em Foco
**Pasta:** projetos/exemplo

## Instruções
- Altere o campo "Pasta" acima para apontar para o roteiro em que está trabalhando.
- Todos os comandos e protocolos do framework usarão os arquivos dentro dessa pasta.
- Para criar um novo roteiro, copie `framework/templates/` para `projetos/<nome>/` ou execute `./novo-roteiro.sh <nome>`.

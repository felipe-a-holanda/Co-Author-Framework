# INSTRUÇÃO AO AGENTE

Antes de qualquer edição ou geração de texto:

1. Consultar status.md para verificar nível de liberdade.
2. Identificar modelo estrutural ativo em estrutura.md.
3. Revisar tema.md para alinhar conflito ideológico.
4. Consultar personagens.md para manter coerência psicológica.
5. Verificar continuidade.md para evitar inconsistências.
6. Confirmar que a cena altera algum estado narrativo.
7. Em caso de dúvida estrutural, propor alternativa no chat antes de modificar.

## Princípios

- Nenhuma cena é neutra.
- Todo diálogo deve carregar objetivo oculto.
- Toda ação deve gerar consequência.
- Evitar exposição direta quando subtexto for possível.
- Conflito > explicação.

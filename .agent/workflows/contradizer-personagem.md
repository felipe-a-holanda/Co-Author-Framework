# /contradizer-personagem

1. Perguntar ao usuário qual personagem contradizer (ou inferir pelo contexto da cena atual).
2. Ler `atlas/personagens.md` e localizar:
   - Contradições Essenciais
   - Crença Limitante
   - Máscara Social
   - Ferida Original
3. Ler a cena atual para identificar o comportamento esperado do personagem.
4. Gerar uma ação ou fala que:
   - Contradiga a máscara social do personagem.
   - Seja coerente com sua ferida original ou necessidade interna.
   - Surpreenda o leitor mas pareça inevitável em retrospecto.
5. Verificar coerência com `atlas/continuidade.md`.
6. Apresentar a variação ao usuário para aprovação.

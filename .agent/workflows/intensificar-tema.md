# /intensificar-tema

1. Ler `atlas/tema.md` para extrair:
   - Pergunta Dramática Central
   - Tese vs Antítese
   - Tensões Secundárias
   - O que NÃO queremos dizer
2. Ler a cena atual indicada pelo usuário.
3. Avaliar: a cena está servindo como argumento dramático ou apenas como evento?
4. Reescrever a cena para que:
   - O conflito entre personagens espelhe a tensão tese vs antítese.
   - Pelo menos uma tensão secundária esteja presente.
   - A cena avance a pergunta dramática sem respondê-la de forma simplista.
5. Garantir que NÃO caia nos erros listados em "O que NÃO queremos dizer".
6. Apresentar a versão intensificada ao usuário.

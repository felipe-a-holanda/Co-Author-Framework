# /intensificar-tema

1. Ler `projeto-ativo.md` para identificar o roteiro ativo (`{projeto}`).
2. Ler `{projeto}/atlas/tema.md` para extrair:
   - Pergunta Dramática Central
   - Tese vs Antítese
   - Tensões Secundárias
   - O que NÃO queremos dizer
3. Ler a cena atual indicada pelo usuário.
4. Avaliar: a cena está servindo como argumento dramático ou apenas como evento?
5. Reescrever a cena para que:
   - O conflito entre personagens espelhe a tensão tese vs antítese.
   - Pelo menos uma tensão secundária esteja presente.
   - A cena avance a pergunta dramática sem respondê-la de forma simplista.
6. Garantir que NÃO caia nos erros listados em "O que NÃO queremos dizer".
7. Apresentar a versão intensificada ao usuário.

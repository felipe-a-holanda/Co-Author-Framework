# /subtexto-maximo

1. Ler `projeto-ativo.md` para identificar o roteiro ativo (`{projeto}`).
2. Ler a cena atual indicada pelo usuário.
3. Ler `{projeto}/atlas/personagens.md` para revisar:
   - Objetivo Externo vs Necessidade Interna de cada personagem presente.
   - Máscara Social e Contradições.
4. Identificar diálogos onde personagens dizem exatamente o que pensam ou sentem.
5. Reescrever esses diálogos para que:
   - O personagem diga uma coisa mas comunique outra.
   - A tensão entre objetivo externo e necessidade interna apareça nas entrelinhas.
   - O leitor/espectador entenda mais do que os personagens dizem.
6. Verificar que a informação dramática essencial ainda é transmitida.
7. Apresentar a versão com subtexto ao usuário.

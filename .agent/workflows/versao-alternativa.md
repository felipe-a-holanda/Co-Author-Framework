# /versao-alternativa

1. Ler a cena atual indicada pelo usuário.
2. Consultar `atlas/estrutura.md` para identificar a função dramática da cena:
   - Função Dramática
   - Mudança de Estado
   - Informação Revelada
3. Gerar uma versão alternativa que:
   - Mantenha a mesma função dramática e mudança de estado.
   - Altere abordagem, tom, ponto de vista ou sequência de eventos.
   - Preserve a coerência com `atlas/personagens.md` e `atlas/continuidade.md`.
4. Salvar a versão alternativa em `rascunhos/` com nome descritivo.
5. Apresentar ambas as versões lado a lado para o usuário escolher.

# /versao-alternativa

1. Ler `projeto-ativo.md` para identificar o roteiro ativo (`{projeto}`).
2. Ler a cena atual indicada pelo usuário.
3. Consultar `{projeto}/atlas/estrutura.md` para identificar a função dramática da cena:
   - Função Dramática
   - Mudança de Estado
   - Informação Revelada
4. Gerar uma versão alternativa que:
   - Mantenha a mesma função dramática e mudança de estado.
   - Altere abordagem, tom, ponto de vista ou sequência de eventos.
   - Preserve a coerência com `{projeto}/atlas/personagens.md` e `{projeto}/atlas/continuidade.md`.
5. Salvar a versão alternativa em `{projeto}/rascunhos/` com nome descritivo.
6. Apresentar ambas as versões lado a lado para o usuário escolher.

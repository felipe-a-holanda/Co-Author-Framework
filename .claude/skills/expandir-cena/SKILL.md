# /expandir-cena

1. Ler `projeto-ativo.md` para identificar o roteiro ativo (`{projeto}`).
2. Ler `{projeto}/atlas/tema.md` para identificar a pergunta dramática central.
3. Ler `{projeto}/atlas/personagens.md` para revisar contradições e necessidades internas dos personagens presentes na cena.
4. Ler a cena atual no arquivo indicado pelo usuário (ou a última cena editada conforme `{projeto}/status.md`).
5. Identificar momentos onde o conflito é superficial ou ausente.
6. Reescrever a cena adicionando:
   - Subtexto nos diálogos (o que o personagem quer vs o que diz).
   - Tensão entre personagens baseada em suas contradições essenciais.
   - Conflito latente que antecipe eventos futuros.
7. Verificar coerência com `{projeto}/atlas/continuidade.md`.
8. Apresentar a versão expandida ao usuário para aprovação.
